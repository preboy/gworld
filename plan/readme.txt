用法：	  export jl filename.xls sheet

jl：      代表导出josn、lua
filename: 要导出的excel文件
sheet:	  要导出的excel中的表名，默认为第一个sheet 也是导出文件名

类型: number float map string array array_string
示例：sample.xlsx



-------------------------------------------------------------------------------
关于有百分比相关属性特别说明

属性加成（类型为2）、暴击率、暴击伤害，附加属性中的百分比等等，都是会在程序中除以100，配置表中配置整数就可以了
